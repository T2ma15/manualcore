# Demanda de documentos en RD y LATAM — scope del MVP (jun-2026)

Investigación verificada (6 frentes: mercado RD, PyMEs LATAM, implementadores ERP,
calidad/ISO, ranking de demanda, MVP-vs-premium). Resumen accionable.

## Conclusión (headline)
SOP + Flujograma = núcleo MVP correcto y validado (los únicos que TODO comprador
quiere y que se venden **sin necesidad de certificación**). Lo de mayor ROI que
faltaba: **Manual de Funciones / Descripción de Puesto** (casi gratis de construir —
cada paso ya tiene un dueño — y dolor nombrado que ningún competidor incluye). BPMN,
planes de inspección y los packs sanitarios = premium/vertical. Política/objetivos/
alcance de calidad: NO son gancho; incluir solo como extra gratis, nunca como pitch,
nunca vender "ISO-ready" ni "QMS".

## MVP recomendado (orden)
1. **SOP / Procedimiento** (plantillas distintas Manufactura y Administrativo). El
   único documento que pide cada comprador (ISO, ERP, homologación de proveedores,
   licitación pública, y dolor puro de escalar/onboarding). Monetiza sin certificación.
2. **Flujograma de proceso** (ANSI simple). Compañero visual universal del SOP, el
   artefacto más "demostrable" en una venta. Mapea al entregable AS-IS del ERP.
3. **Manual de Funciones / Descripción de Puesto + Organigrama** — MAYOR ROI no
   considerado. Sale de la MISMA entrevista (cada paso tiene responsable). Dolor
   frecuente en RD ("Manual de Cargos y Funciones" es el estándar del sector público).
4. **Caracterización / Ficha de Proceso** — expectativa universal LATAM, punto ciego;
   tejido conectivo de gestión por procesos (entradas/salidas/recursos/indicadores/
   dueño). Barato de plantillar; se vende suelto en kits ISO.

## Premium (versiones siguientes)
- **Programa de SST (Reglamento 522-06) con matriz de riesgo / IPERC** — PRIMER doc de
  pago y el mayor forzador legal: 522-06 obliga a TODO empleador de RD (cualquier
  tamaño/sector) a tener un programa SST escrito, inspeccionable por Min. Trabajo.
  Actualización anual = ingreso recurrente. Aquí vive el "análisis de riesgo".
- **BPMN (AS-IS / TO-BE) como MODO de salida del flujograma** para implementadores ERP.
  Comprador B2B de mayor presupuesto (CO/MX > RD). Diferenciador = GENERACIÓN automática
  desde entrevista informal, no el lienzo (Bizagi ya existe gratis). Fast-follow, no día 1.
- **Pack sanitario: Manual BPM + POE/POES** (alimentos/farma/cosmética). Licencia para
  operar: DIGEMAPS/MSP (Decreto 246-06) lo exige para registro sanitario. Lleva CCP →
  diferir hasta que el manejo de criticidad esté probado.
- **Plan HACCP** (alimentos/agro-exportación: mango, cacao). CCP-aware.
- **Plan de Inspección y Ensayo / especificación de inspección de producto** — necesidad
  real pero el universo MÁS PEQUEÑO (solo manufactura/zona franca/agro-export; RD es
  94.2% micro de servicios/comercio). Plantilla DISTINTA (característica × método × AQL ×
  frecuencia × registro). Venderlo al vertical industrial junto con BPM/HACCP.
- **Dossier ISO 9001 / NORDOM 774 (auto-compilado) + Lista Maestra**. Output premium que
  se arma de los SOP ya cargados. NORDOM 774 (norma MIPYME RD, fondo MICM-EU) = buen
  gancho narrativo. Nunca pitch "ISO-ready"/"QMS".
- **Capa de analítica / mejora sobre la data** (VSM, Gantt, cuellos de botella, FMEA,
  Control Plan). El valor depende de data ya cargada → lock-in. FMEA/Control Plan =
  jugada México-automotriz, casi nula demanda RD.

## Veredicto sobre lo que Tania propuso
- **SOP** → MVP núcleo (validado #1).
- **Flujograma** → MVP núcleo (validado #2).
- **BPMN** → premium (comprador más sofisticado/estrecho; construir como modo, no producto).
- **Plan de inspección** → premium/vertical (universo más pequeño; distinto del WI).
- **Análisis de riesgo** → MVP add-on PERO en sabor **SST/ocupacional (522-06, IPERC)**,
  que tiene mandato legal para TODOS los empleadores y gancho de actualización anual.
  El riesgo empresarial abstracto (AMEF/ISO 31000) tiene poca tracción orgánica → premium.
  NO fusionar los dos sabores.
- **Política + objetivos + alcance de calidad** → drop como gancho; incluir solo como
  quick-win gratis auto-generado junto al SOP (los objetivos ganan valor cuando se vuelven
  indicadores medibles que alimentan el dashboard premium).
